# iPhone Notification Simulator – Live Lock Screen Notification Preview Tool

A free, open-source **iPhone-style notification simulator** built with pure **HTML, CSS, and JavaScript** — no frameworks, no dependencies. Design, customize, and preview realistic **iOS lock screen notifications** and **notification stacks/bundles** directly in your browser, then play them back with smooth scroll animations.

🔗 **[Live Demo](#)** &nbsp;|&nbsp; 📦 **No build tools required** &nbsp;|&nbsp; ⚡ **Single HTML file**

---

## ✨ Features

- 📱 **Realistic iPhone lock screen UI** – live clock, date, and dark gradient wallpaper
- 🔔 **Custom notification builder** – set app icon, icon color, title, message, time label, and badge count
- 🗂️ **Notification stack / bundle preview** – grouped notifications with a "+N more" counter, just like iOS
- 🎬 **Smooth bottom-to-top scroll animation** – new notifications slide in and shift the stack upward with no flicker (FLIP animation)
- ⏱️ **Full playback controls** – customizable start delay, speed between notifications, auto-dismiss timer, loop mode, and max visible cards
- 🎨 **Theme customization** – adjustable lock screen background gradient, plus **glass (blurred/translucent)** or **normal (solid)** notification card styles with opacity control
- 🧩 **Queue management** – add, reorder, remove, and clear notifications before running the simulation
- 🚀 **Zero dependencies** – works by simply opening `index.html`, perfect for GitHub Pages

## 🚀 Getting Started

### Option 1: Run locally
1. Clone or download this repository
2. Open `index.html` in any modern browser
3. Customize your notification on the left panel and press **Start**

```bash
git clone https://github.com/dizetech/iphone-notification-simulator.git
cd iphone-notification-simulator
open index.html
```

### Option 2: Host on GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Select the `main` branch and `/ (root)` folder
4. Your live demo will be available at:
   `https://dizetech.github.io/iphone-notification-simulator/`

## 🛠️ Customization Options

| Setting | Description |
|---|---|
| Icon, icon color, title, message, time, count | Build a custom notification |
| Background gradient | Two-color picker for lock screen wallpaper |
| Notification style | Glass (blur/translucent) or normal (solid) |
| Glass opacity | Fine-tune transparency level |
| Start delay | Delay before the first notification appears |
| Speed | Time between each notification |
| Auto-dismiss | Automatically hide notifications after N seconds |
| Loop queue | Replay the notification queue continuously |
| Max visible | Number of notification cards shown at once |

## 🧱 Tech Stack

- **HTML5** – semantic structure
- **CSS3** – flexbox layout, CSS variables, keyframe animations, glassmorphism (`backdrop-filter`)
- **Vanilla JavaScript** – DOM manipulation, FLIP animation technique, timers/queues

No React, no build step, no npm install — just open and run.
